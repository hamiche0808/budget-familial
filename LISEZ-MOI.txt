  INSTALLER SUR LE BUREAU WINDOWS (icône + raccourci)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  Dans Chrome ou Edge :
  1. Lancez l'app avec LANCER.bat
  2. En haut à droite du navigateur, cliquez sur ⁝ (trois points)
  3. Cliquez sur "Installer Budget Familial..."
     ou "Enregistrer et partager" → "Installer en tant qu'application"
  4. Une icône apparaît sur votre bureau !
  5. L'app s'ouvre dans sa propre fenêtre, sans barre de navigateur.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  ACCÈS DEPUIS LE TÉLÉPHONE (même réseau Wi-Fi)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  1. Lancez LANCER.bat sur votre PC
  2. Dans le terminal, notez l'adresse "Tél." affichée
     Exemple : http://192.168.1.42:3000
  3. Sur votre téléphone (même Wi-Fi), ouvrez cette adresse
  4. Dans Chrome mobile : menu ⁝ → "Ajouter à l'écran d'accueil"
  5. L'icône apparaît sur votre téléphone comme une vraie app !

  ⚠️  Votre PC doit rester allumé pour que le téléphone y accède.

  → Désactivez temporairement le pare-feu Windows

  → Utilisez Chrome ou Edge (pas Firefox)
